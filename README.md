# Developer's docker-compose.yml
Docker-compose.yml file for development tools
- PostgreSQL
- MSSql
- MySQL
- Redis
- MongoDB
- Zookeeper
- Kafka
- Krakend

Up All Services
```
docker compose up -d
```

Up Specific Service
```
docker compose up -d postgres
```

Down All Services
```
docker compose down
```

Down Specific Service
```
docker compose down postgres
```
